Java Documentation

The following documentation has been generated form the ar:kalix source code:

kalix-base Foundational classes and utilities required to create Arrowhead systems. If reading the documentation for the first time, this is the place to start at.
kalix-core-plugins Plugins useful for integration against the Arrowhead Framework core services.
kalix-processors Internal API. Annotation processors, used to automatically generate parsers and generators for data transfer objects. Note that the code described here will never be invoked directly by Kalix users. The Javadoc is included here, however, for the sake of completeness.

Note that the work of documenting the source code is on-going. The documentation is currently very much incomplete.
